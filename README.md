# docgenius-mcp

Documentation MCP u2014 aggregate docs from official docs, Stack Overflow, GitHub Issues, dev blogs, tutorials, and code examples. Get daily update summaries.

## Quick Start

```bash
git clone https://github.com/marilynceo/docgenius-mcp.git
cd docgenius-mcp
pip install -r requirements.txt
python src/server.py
```

## Gateway

**Production endpoint:** https://docgenius.zhc-mcp.org

## Tools

See `src/server.py` for full tool list.

## Installation

```bash
# Via Smithery
npx @smithery/cli mcp add marilynceo/docgenius-mcp

# Or connect directly via MCP client
# Endpoint: https://docgenius.zhc-mcp.org/mcp
```

## Configuration

No API keys required. Server runs locally or via gateway.

## Privacy

All processing happens in-memory. No data stored on servers.
